# Quick Start


### Up Database:

```
docker-compose up -d --build
```



### Install SAP HANA Client:

[Instalação do Client SAP Hana](https://developers.sap.com/tutorials/hana-clients-install.html)


Add in this folder:

1. Linux:
```
/SapHana
```

2. Windows:
```
C:\SapHana
```



### Add seeds
Execute SQL commands in seeds folder.
